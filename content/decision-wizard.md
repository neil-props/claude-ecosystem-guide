### Extension Points at a Glance

If JavaScript is disabled, here is a quick guide to the main extension points:

- **[Skills](skills.html)** -- Reusable markdown instructions that teach Claude repeatable processes
- **[Agents](agents.html)** -- Isolated Claude instances for complex multi-step workflows
- **[Commands](commands.html)** -- Quick slash command shortcuts for common prompts
- **[MCP](mcp.html)** -- Connect Claude to external tools and APIs via the Model Context Protocol
- **[Connectors](connectors.html)** -- Pre-built one-click integrations for popular services
- **[Plugins](plugins.html)** -- Bundled packages of skills, hooks, MCP configs, and agents
- **[Hooks](hooks.html)** -- Automatic code that runs at lifecycle events (pre/post actions)
- **[Projects](projects.html)** -- Project-wide instructions via CLAUDE.md files
- **[Memory](memory.html)** -- Persistent context across sessions
- **[Settings](settings.html)** -- Configure Claude Code behavior at multiple levels

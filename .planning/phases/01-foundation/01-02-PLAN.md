   - Mobile nav button position: fixed top-left, z-index above overlay
   - Hamburger-to-X animation: `.nav-active .hamburger-line:nth-child(1) { transform: rotate(45deg) ... }` etc.
   - `.content` padding adjusts: `padding: 1rem` on mobile

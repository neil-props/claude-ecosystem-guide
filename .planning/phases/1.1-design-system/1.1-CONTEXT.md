<decisions>
## Implementation Decisions

### Design Source
- Use the Props style guide (`~/Desktop/props/props-stye-guide/index.html`) as the authoritative design reference
- Extract and adapt all design tokens — do NOT invent new ones

### Color Palette
- **Primary**: Oxford Blue `#001639`, Isabelline `#EBE6E0`
- **Secondary**: Turquoise `#51DDCD`, Icterine `#FFFA78`
- **Text-only**: `#FEFDFD` (near-white for dark backgrounds)
- **Full tint system**: Oxford (900/700/500), Isabelline (700/300/100), Turquoise (700/300/100), Icterine (700/300/100)
- **Neutrals**: dark-gray `#1a1a1a`, medium-gray `#464646`, muted-gray `#7E899A`

### Typography
- **Serif**: IBM Plex Serif (Display, H1-H4, emphasis/pull quotes)
- **Sans**: Inter (body, subheaders, UI elements)
- **Fluid scale** using `clamp()` — from `--text-display` (4-7rem) down to `--text-xs` (0.75rem)
- **Code**: 'SF Mono', 'Fira Code', 'Consolas', monospace

### Spacing
- **Fluid 8-step scale** using `clamp()` — `--space-1` through `--space-8`
- **Section spacing**: `--section-sm`, `--section-md`, `--section-lg`, `--section-page-top`
- **Sidebar width**: 260px
- **Container max**: 1120px

### Border Radius
- sm: 0.5rem, md: 1rem, lg: clamp(1.5-2rem), xl: clamp(2-3.125rem), full: 100px (pills)

### Shadows (4-tier elevation)
- sm/md/lg/xl using Oxford Blue rgba

### Transitions
- fast: 150ms, base: 300ms, slow: 500ms

### Dark/Light Theme
- **Dark**: Oxford Blue background, text-only text, turquoise/icterine accents
- **Light**: Isabelline background, Oxford Blue text, turquoise/icterine accents

### Component Patterns to Implement
- **Buttons**: pill-shaped (border-radius: 100px), primary/secondary/outline/ghost/icterine variants
- **Badges**: pill-shaped with optional dot indicator, brand + status variants
- **Cards**: radius-lg, shadow-md, hover elevation
- **Tables**: Oxford Blue header, white rows, isabelline-300 hover
- **Code blocks**: Oxford Blue bg, syntax highlighting (turquoise keywords, icterine strings)
- **Lists**: turquoise bullet dots (8px), numbered lists with turquoise circular badges (28px)
- **Navigation links**: animated underline on hover (0→100% width)
- **Sidebar**: Oxford Blue bg, turquoise active indicator (3px left border)
- **Eyebrow labels**: Inter, uppercase, 0.2em letter-spacing, turquoise/icterine/oxford variants

### Sidebar Design
- Width: 260px, Oxford Blue background
- Logo: serif, 1.75rem, turquoise, 0.15em letter-spacing, horizontal rules above/below
- Links: text-only color, hover gets 3px turquoise left border + subtle bg
- Hidden on mobile <870px

### Responsive Breakpoints
- sm: 480px, md: 768px, lg: 992px
- Mobile: hide sidebar, single-column, reduced spacing

### Content Layer UI (Tabs/Accordions)
- Use the component patterns to create tab UI for concept/howto/reference/decision-tree views
- Tab style should follow the nav link pattern (turquoise active indicator)

### Interactive Elements
- Decision wizard: card-based with turquoise active states
- Search results: card-based with badges for result type
- Code blocks: Oxford Blue bg, copy button with turquoise hover
- Copy feedback: brief turquoise flash or "Copied!" badge

### Mobile Navigation
- Hamburger: 3 lines, 1.5rem wide, 2px thick, 0.4rem gap
- Open state: rotation animation (45°/135°), 300ms ease
- Drawer: slides from left, Oxford Blue bg

### Gradients Available
1. Oxford Deep: 135deg #001639→#002859
2. Oxford Bright: 135deg #001639→#004FAB
3. Turquoise Fade: 135deg #37CABD→#51DDCD
4. Turquoise to Icterine: 135deg #51DDCD→#FFFA78
5. Oxford to Turquoise: 135deg #001639→#51DDCD
6. Isabelline Soft: 135deg #EBE6E0→#FEFDFD

### Claude's Discretion

- Tab/accordion implementation details for the layered content depth model
- Search results UI layout
- Decision wizard step-by-step flow UI

<specifics>
## Specific Ideas

- The Props style guide is a single 98KB HTML file at `~/Desktop/props/props-stye-guide/index.html`
- All design tokens are CSS custom properties in `:root` scope — highly portable
- Syntax highlighting in code blocks should use: turquoise (keywords), icterine (strings), muted-gray (comments), turquoise-300 (functions), #FF8A65 (numbers)
- All colors meet WCAG AA contrast standards (Oxford Blue on white: 18.91:1)
- Accessibility: focus rings use turquoise, 2px solid with 3px offset
- Font smoothing: `-webkit-font-smoothing: antialiased`

</specifics>

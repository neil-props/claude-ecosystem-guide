Claude Ecosystem Guide#

Welcome to the comprehensive guide to the Claude Code ecosystem. This site covers all the extension points and configuration options available in Claude Code, helping you understand which tools to use and how to set them up.

Extension Points#

Explore the 10 core extension points that make Claude Code customizable:

  • MCP Servers and Connectors - Connect Claude to external tools and data sources
  • Skills - Teach Claude reusable patterns and capabilities
  • Plugins - Extend Claude Code with third-party integrations
  • Hooks - Run scripts at key lifecycle events
  • Agents - Create specialized AI workflows
  • Projects and CLAUDE.md - Configure project-level behavior
  • Connectors - Connect Claude to external services
  • Commands - Define reusable prompt shortcuts
  • Memory - Manage Claude's persistent context
  • Settings - Control Claude Code configuration at every level
